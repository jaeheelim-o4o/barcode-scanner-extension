# 설계서 v2 — Barcode Scanner Extension 기능 확장

- **작성일**: 2026-06-22
- **버전**: v2.0 (v1.0 기능 확장)
- **추가 기능**: 매장 선택 + 바코드 목록 + localStorage 설정 + 즐겨찾기

---

## 1. 추가 요구사항

| # | 요구사항 | 설명 |
|---|----------|------|
| ① | 매장 선택 → 테스트 바코드 목록 | PROD 매장 선택 시 상품명+바코드 목록 표시, 클릭해서 입력 |
| ② | localStorage 자동 설정 | 매장 선택 시 `local-pos-id` 값 자동으로 현재 탭에 설정 |
| ③ | 즐겨찾기 | 자주 쓰는 매장/상품을 chrome.storage에 저장하여 빠른 접근 |

---

```js

    }

```

## 4. 메시지 프로토콜 확장

| type | payload | 동작 |
|------|---------|------|
| `SCAN_BARCODE` | `{ barcode, delay }` | 기존 키보드 입력 시뮬레이션 |
| `SET_LOCAL_STORAGE` | `{ key, value }` | `localStorage.setItem(key, value)` |
| `GET_LOCAL_STORAGE` | `{ key }` | `localStorage.getItem(key)` 반환 |
