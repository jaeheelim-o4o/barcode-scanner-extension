# 설계서 v4 — local-pos-id 편집 및 활성 표시

- **작성일**: 2026-06-22
- **버전**: v4.0 (v3.0 기능 확장)
- **추가 기능**: 커스텀 매장 localPosId 인라인 편집 + 내 매장 탭 현재 설정값 강조 표시

---

## 1. 추가 요구사항

| # | 요구사항 |
|---|----------|
| ① | 커스텀 매장의 local-pos-id 값을 직접 수정 가능 |
| ② | 여러 매장을 등록해두고 클릭 한 번으로 pos-id 전환 |
| ③ | 내 매장 탭에서 현재 설정 중인 매장이 어디인지 시각적으로 표시 |

---

## 2. 변경 내용

### 2-1. 커스텀 매장 localPosId 인라인 편집

**현재**: pos-info 영역에서 localPosId가 텍스트로만 표시됨  
**변경**: pos-id 텍스트를 클릭하면 `<input>`으로 전환 → Enter 또는 [저장] 클릭 시 customStores에 반영 + localStorage에도 즉시 적용

```
[ local-pos-id ]  [ 66-5 (클릭 편집) ]  [설정하기]
                         ↓ 클릭
[ local-pos-id ]  [ _66-3________ ]  [저장]  [취소]
```

- 커스텀 매장 선택 시에만 pos-id 텍스트가 클릭 가능 (기본 매장은 편집 불가)
- 저장 시: `customStores` 업데이트 + `localStorage['local-pos-id']` 즉시 반영 + 드롭다운 갱신

### 2-2. 내 매장 탭: 현재 설정 중인 매장 하이라이트

**현재**: 모든 카드가 동일하게 표시됨 — 어느 매장이 활성화된지 알 수 없음  
**변경**: 팝업 열릴 때 localStorage에서 현재 `local-pos-id`를 읽어 일치하는 카드에 "활성" 배지 표시

```
┌─────────────────────────────────────────┐
│  ⭐ 66-5 매장  66-5  🟢 현재 설정 중  [✕]  │  ← 강조
└─────────────────────────────────────────┘
┌─────────────────────────────────────────┐
│  ⭐ 77-3 매장  77-3             [설정]  [✕]  │
└─────────────────────────────────────────┘
```

- 활성 카드: 배경 연녹색 + "현재 설정 중" 배지 (초록)
- 비활성 카드: 기존과 동일 + [설정] 버튼 표시
- [설정] 클릭 시 카드 하이라이트 즉시 갱신 (re-render)
