# 설계서 v3 — 커스텀 매장 관리 기능

- **작성일**: 2026-06-22
- **버전**: v3.0 (v2.0 기능 확장)
- **추가 기능**: 커스텀 매장 추가 → 드롭다운 노출 + 이름 편집 + 바코드 수동 추가

---

## 1. 추가 요구사항

| # | 요구사항 |
|---|----------|
| ① | 즐겨찾기 추가 시 매장 탭 드롭다운에도 나타남 |
| ② | 추가한 매장 이름 편집 가능 |
| ③ | 추가한 매장에 테스트 바코드 수동 추가 가능 |

---

## 2. 데이터 설계 변경

### 2-1. 현재 (v2) 데이터 구조

```
STORES (하드코딩) → 매장 탭 드롭다운
chrome.storage.local.favorites → 즐겨찾기 탭
```

두 목록이 **분리**되어 있어 즐겨찾기 추가해도 드롭다운에 안 나타남.

### 2-2. v3 데이터 구조

```
chrome.storage.local.customStores → 사용자가 추가한 매장
STORES (하드코딩) + customStores → 매장 탭 드롭다운 (합산)
즐겨찾기는 customStores와 통합 (별도 favorites 불필요)
```

### 2-3. customStores 스키마

```js
{
  customStores: [
    {
      id: 'custom_1750000000000',   // 고유 ID
      isCustom: true,               // 하드코딩 STORES와 구분
      name: '66-5 매장',            // 편집 가능
      description: '',
      localPosId: '66-5',
      products: [                   // 수동 추가한 바코드
        { name: '상품명', uid: '', barcodes: ['BARCODE001'] }
      ]
    }
  ]
}
```

## 3. UI 설계

### 3-1. 매장 탭 드롭다운

```
[PROD] 무신사 편집샵 (shopNo: 25) — 무신사 스토어 성수@대림창고
[PROD] 무신사 스탠다드 (shopNo: 11) — 무신사 스탠다드 성수점
[PROD] 29CM (shopNo: 49) — 29CM 현대백화점 판교
[PROD] 29CM (shopNo: 51) — 29CM 이구홈 성수
──────────────────────────── (구분선)
⭐ 66-5 매장   ✎ 편집   ✕ 삭제     ← 커스텀 매장 (편집/삭제 버튼 노출)
⭐ 77-3 매장   ✎ 편집   ✕ 삭제
```

### 3-2. 커스텀 매장 선택 시

- local-pos-id 표시 + [설정하기]
- 상품 목록 + **[+ 바코드 추가]** 버튼
- 바코드 추가 인라인 폼: `상품명` | `바코드` | [추가] / [취소]

```
┌──────────────────────────────────────┐
│  ⭐ 66-5 매장                         │
│  local-pos-id: 66-5    [설정하기]     │
│                                       │
│  ── 테스트 상품 목록 ──                │
│  ▶ 나이키 에어맥스     #-    [삭제]   │
│    BARCODE001          [입력]         │
│                                       │
│  [+ 바코드 추가]                      │
│  ┌────────┐ ┌────────────────┐        │
│  │ 상품명  │ │ 바코드          │ [추가] │
│  └────────┘ └────────────────┘        │
└──────────────────────────────────────┘
```

### 3-3. 이름 편집

드롭다운 옆 ✎ 버튼 → 인라인 텍스트 편집 → 저장

## 4. 파일 변경 계획

| 파일 | 변경 내용 |
|------|-----------|
| `data/stores.js` | 변경 없음 (하드코딩 STORES 유지) |
| `popup/popup.js` | customStores 로드/저장, 드롭다운 통합 렌더링, 바코드 추가/삭제/이름편집 |
| `popup/popup.html` | 바코드 추가 인라인 폼 영역 추가 |
| `popup/popup.css` | 편집 폼, 커스텀 매장 배지 스타일 |
